## 最需要調整的熱點名稱

## 我覺得最不吸引人的類型排序

| 優先修正順序 | 類型                       | 原因                |
| -----: | ------------------------ | ----------------- |
|      1 | 沒有對戰資訊的「全場精華／全場數據／上半場精華」 | 使用者完全不知道是哪場       |
|      2 | 「正賽開始／比賽開始」              | 不像精華，不構成回看理由      |
|      3 | 「攻守數據／進攻路線比較」            | 偏分析報表語言，缺少娛樂性     |
|      4 | 英文大寫球員名＋無隊伍情境            | 不熟球員的用戶看不懂        |
|      5 | 梗感太重或語意怪的標題              | 可能有趣，但不穩定，容易降低專業感 |
|      6 | 標點或空格造成的近似重複             | 影響列表品質，也會干擾統計     |

## 彙整原因

這批熱點名稱的主要問題不是「沒有精彩片段」，而是 **標題沒有把精彩點翻譯成用戶想點的理由**。
尤其是「全場數據」、「正賽開始」、「全場比賽精華」這類命名，對後台或剪輯流程有意義，但對使用者來說資訊量太低。

比較理想的熱點名稱應該至少包含三個元素中的兩個：

> 對戰組合／球星或隊伍／事件結果

例如：

| 原本                   | 建議方向                    |
| -------------------- | ----------------------- |
| 全場比賽精華               | 葡萄牙驚險晉級｜全場精華            |
| 全場數據                 | 阿根廷壓制約旦｜全場攻守數據          |
| 正賽開始                 | 開賽 3 分鐘即射正，巴拿馬率先施壓      |
| 阿爾GOUIRI再下一城 角球好用一直用 | 阿爾及利亞角球奏效，Gouiri 頭槌再下一城 |
| VAR升堂明察秋毫揪出奧地利手球犯規   | VAR 介入改判，奧地利禁區手球送點      |

我的結論是：
**目前最該修的是「泛用標題」與「資料型標題」，其次才是球員譯名或文案風格。** 因為前者會直接讓用戶不知道內容是什麼，對點擊率傷害最大。
